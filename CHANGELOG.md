## [0.7.0] - 2025-10-30

### Added
- Comprehensive README with detailed documentation
- MIT License
- Contributing guidelines
- GitHub issue and PR templates
- Support for multiple LLM backends (Ollama, OpenAI)
- Multi-agent orchestration system
- Blockchain integration with Antelope.io
- IPFS decentralized storage support
- Text-to-image generation with FLUX models
- Image editing capabilities
- Comprehensive NLP metrics suite (BLEU, ROUGE, METEOR, etc.)
- MongoDB integration for result storage
- ChromaDB vector database integration
- Batch testing capabilities
- Real-time progress tracking with WebSockets
- Interactive results visualization with Chart.js
- CSV and JSON export functionality

### Features
- RAG system implementation with configurable retrieval
- Document processing (PDF, DOCX, TXT, Web)
- Multi-agent pipeline testing
- Performance metrics tracking
- Token usage monitoring

- Wharf wallet integration

### Documentation
